设备广播名字为UARTF

测试步骤：
一、微信运动测试
1、将开发连接USB，电脑打开串口工具，波特率：115200

2、使用微信测试AirSyncDebug扫描设备，并选择设备

3、选择“精简协议”->“计步器测试”->自动测试到设置运动目标

4、按下开发板的KEY3，设备上报计步数据，设备通知目标变化

5、测试完成，断开连接

二、ANCS与HID测试
1、使用IOS手机，在手机系统的蓝牙里面选择连接UARTF设备

2、设备发起请求配对，手机同意配对

3、按下开发板的KEY1是音量加；按下开发板的KEY2是音量减

4、给IOS的QQ、微信等APP发送数据，进行测试，串口助手打印ANCS数据

5、使用手机给IOS设备打电话，按下开发板的KEY4接电话

6、测试完成
